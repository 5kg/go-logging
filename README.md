#go-logging
The logging package for golang

It supports the following formats
```
	name            string
	seqid           uint64
	levelno         int
	levelname       string
	pathname        string
	filename        string
	lineno          int
	funcName        string
	created         int64
	asctime         string
	msecs           int64
	relativeCreated int64
	thread          int
	threadName      string
	process         int
	message         string
	timestamp       int64
```
